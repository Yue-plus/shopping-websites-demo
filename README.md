# 九职网页设计软件 2201 作业 | 设计购物网站

## 运行

开始之前确保已经安装最新版的 [Node.js](https://nodejs.org/zh-cn/)：

```shell
$ node -v
v18.12.0
```

**使用 Yarn 时：**

```shell
yarn
yarn start
```

** 使用 NPM 时：**

```shell
npm i
npm run start
```
